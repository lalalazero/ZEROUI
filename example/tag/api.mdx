## API

### Tag 的属性设置如下

| 属性      | 说明           | 类型       | 可选值 | 默认值 | 是否必填 |
| --------- | -------------- | ---------- | ------ | ------ | -------- | --- |
| visible   | 是否显示       | boolean    | --     | --     | --       | --  |
| closeable | 是否可关闭     | boolean    | --     | --     | --       | --  |
| onClose   | 关闭的回调函数 | () => void | --     | --     | --       | --  |
| color     | tag 的颜色设置 | string     | --     | --     | --       | --  |

### CheckableTag 的属性设置如下

| 属性     | 说明           | 类型                       | 可选值 | 默认值 | 是否必填 |
| -------- | -------------- | -------------------------- | ------ | ------ | -------- | --- |
| checked  | 是否选中       | boolean                    | --     | --     | --       | --  |
| onChange | 选中变化的回调 | (checked: boolean) => void | --     | --     | --       | --  |
