## 栅格排序

通过使用 `push` 和 `pull` 改变列的顺序。

```jsx
import React from 'react'
import ReactDOM from 'react-dom'
import { Row, Col } from 'zero-ui-react'

const App = () => {
    return (
        <div className="col-with-bg">
            <Row>
                <Col span={18} push={6}>
                    col-1 col-span-18-push-6
                </Col>
                <Col span={6} pull={18}>
                    col-2 col-span-6-pull-18
                </Col>
            </Row>
        </div>
    )
}
export default App
```

```css
.zeroUI-row {
    margin: 20px 0;
    color: #fff;
}
.zeroUI-col {
    height: 40px;
    line-height: 40px;
    text-align: center;
}
.zeroUI-col:nth-child(even) {
    background-color: #87e8de;
}
.zeroUI-col:nth-child(odd) {
    background-color: #36cfc9;
}
```
