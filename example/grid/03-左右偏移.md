## 左右偏移

利用 `offset` 实现向右偏移，比如 `offset={4}` 就是向右偏移了4个宽度（宽度总是24份等分）。

```jsx
import React from 'react'
import ReactDOM from 'react-dom'
import { Row, Col } from 'zero-ui-react'

const GridExample = () =>{
    return (
        <div>
            <Row>
                <Col span={6} >col-span-6</Col>
                <Col span={6} >col-span-6</Col>
                <Col span={6} offset={6}>col-span-6-offset-6</Col>
            </Row>
            <Row>
                <Col span={6} >col-span-6</Col>
                <Col span={6} offset={6}>col-span-6-offset-6</Col>
                <Col span={6} >col-span-6</Col>
            </Row>
        </div>
    )
}
export default GridExample
```

```css
.zeroUI-row {
    margin: 20px 0;
    color: #fff;
}
.zeroUI-col {
    height: 40px;
    line-height: 40px;
    text-align: center;
    
}
.zeroUI-col:nth-child(even) {
    background-color: #87e8de;
}
.zeroUI-col:nth-child(odd) {
    background-color: #36cfc9;
}
```
