## API

属性如下：

| 属性      | 说明                         | 类型                                              | 可选值 | 默认值 | 是否必填 |
| --------- | ---------------------------- | ------------------------------------------------- | ------ | ------ | -------- |
| count     | 消息数                       | number                                            | --     | --     | --       |
| className | 样式类名                     | string                                            | --     | --     | --       |
| showZero  | 消息为 0 是否显示徽章        | boolean                                           | --     | false  | --       |
| overCount | 封顶数字,超过后显示为封顶+号 | number                                            | --     | 99     | --       |
| dot       | 显示为小红点                 | boolean                                           | --     | --     | --       |
| offset    | 徽章离右上角的偏移坐标       | [number/string, number/string]                    | --     | --     | --       |
| status    | 小圆点状态                   | 'success'/'waring'/'default'/'processing'/'error' | --     | --     | --       |
| text      | 小圆点状态附加文字说明       | string                                            | --     | --     | --       |
| color     | 自定义小圆点状态颜色         | string                                            | --     | --     | --       |
